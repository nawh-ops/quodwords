# Contributing to QuodWords

Thanks for your interest in QuodWords.

This repository contains the reference specification and supporting
tooling for the QuodWords encoding system. The primary goals are
determinism, reproducibility, and long-term stability.

## Scope

At this stage, contributions are limited to:

- Clarifications or corrections to documentation.
- Bug fixes to verification or tooling scripts.
- Well-justified proposals for future versions (clearly out of scope for v0).

Changes that affect encoding rules, wordlists, or determinism **will not**
be accepted without an explicit version bump.

## Process

- Open an issue to discuss any non-trivial change before submitting code.
- Keep pull requests small, focused, and well-documented.
- Follow the existing structure and conventions.

By contributing, you agree that your contributions may be incorporated
into the project under the existing license.
