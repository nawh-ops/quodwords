# Wordlists

This directory contains the canonical wordlists used by QuodWords.

## v0 policy

- Wordlists are plain text, one word per line.
- Words are lowercase ASCII (`a-z`), with no spaces, punctuation, or diacritics.
- Wordlists are sorted and contain no duplicates.
- Once published, a wordlist file is immutable. Changes require a new versioned file (e.g. `en_v1.txt`).

Current canonical v0 wordlist: `en_v0.txt`.
